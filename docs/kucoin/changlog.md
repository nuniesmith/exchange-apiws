### 2026.03.04

**SPOT Margin HF Migration Completed**  
Parameter Unification & LF Endpoint Deprecation

> Note: To understand the background and migration plan for this change, please read KuCoin’s previously published announcement “KuCoin Margin API Upgrade Announcement”:
> https://www.kucoin.com/announcement/kucoin-margin-notice-250702

KuCoin Margin APIs have completed the high-frequency (HF) migration. All low-frequency (LF) accounts for SPOT Margin have been migrated to HF accounts. Effective immediately, please use HF APIs for SPOT Margin trading and related operations.

**Change Summary**
- **Preferred parameters**: Use non-V2 account type parameters for SPOT Margin (`margin`, `isolated`).
- **V2 parameters**: `margin_v2`, `isolated_v2` have the same meaning as non-V2 parameters, but will be gradually replaced by non-V2 parameters.
- **LF order endpoints disabled**: Some LF Margin order endpoints under Abandoned Endpoints are no longer available.

**Affected Endpoints**

| # | Endpoint | Parameter Mapping & Notes (All refer to SPOT Margin HF accounts) | Change |
|---|---|---|---|
| 1 | Classic REST - [Get Transfer Quotas](https://www.kucoin.com/docs-new/rest/account-info/transfer/get-transfer-quotas.md) | **Preferred**: `MARGIN`, `ISOLATED`<br>**V2**: `MARGIN_V2`, `ISOLATED_V2` (same meaning; phased out gradually) | Queries for **SPOT Margin LF accounts** are no longer supported. |
| 2 | Classic REST - [Flex Transfer](https://www.kucoin.com/docs-new/rest/account-info/transfer/flex-transfer.md) | **Preferred**: `MARGIN`, `ISOLATED`<br>**V2**: `MARGIN_V2`, `ISOLATED_V2` (same meaning; phased out gradually)<br>**Rule**: When `type` is `INTERNAL`, non-V2 and V2 are equivalent. When `type` is not `INTERNAL`, `MARGIN_V2` and `ISOLATED_V2` are not supported—use `MARGIN` and `ISOLATED`. | Transfers **between SPOT Margin LF and HF accounts** are no longer supported. |
| 3 | Classic REST - [Get Account - Cross Margin](https://www.kucoin.com/docs-new/rest/account-info/account-funding/get-account-cross-margin.md) | **Preferred**: `MARGIN`<br>**V2**: `MARGIN_V2` (same meaning; phased out gradually) | Parameter semantics unified: `MARGIN` represents HF Cross Margin; `MARGIN_V2` will be phased out. |
| 4 | Classic REST - [Get Account - Isolated Margin](https://www.kucoin.com/docs-new/rest/account-info/account-funding/get-account-isolated-margin.md) | **Preferred**: `ISOLATED`<br>**V2**: `ISOLATED_V2` (same meaning; phased out gradually) | Parameter semantics unified: `ISOLATED` represents HF Isolated Margin; `ISOLATED_V2` will be phased out. |
| 5 | Classic REST - [Get Account Ledgers - Margin_hf](https://www.kucoin.com/docs-new/rest/account-info/account-funding/get-account-ledgers-marginhf.md) | Response field: `accountType`<br>**Current**: `MARGIN_V2`, `ISOLATED_V2`<br>**Next**: `MARGIN`, `ISOLATED` (will gradually replace the V2 values) | Response field update: `accountType` currently returns V2 enum values and will gradually migrate to non-V2 enum values. |
| 6 | Abandoned Endpoints - [Inner Transfer](https://www.kucoin.com/docs-new/abandoned-endpoints/account-funding/inner-transfer.md) | **Preferred**: `margin`, `isolated`<br>**V2**: `margin_v2`, `isolated_v2` (same meaning; phased out gradually) | Transfers **between SPOT Margin LF and HF accounts** are no longer supported. |
| 7 | Abandoned Endpoints - [Add Order - V1](https://www.kucoin.com/docs-new/abandoned-endpoints/margin-trading/add-order-v1.md) | N/A (Margin **LF** order placement endpoint; no HF parameter mapping) | SPOT Margin LF endpoint is no longer available. |
| 8 | Abandoned Endpoints - [Add Order Test - V1](https://www.kucoin.com/docs-new/abandoned-endpoints/margin-trading/add-order-test-v1.md) | N/A (Margin **LF** test order endpoint; no HF parameter mapping) | SPOT Margin LF endpoint is no longer available. |
| 9 | Abandoned Endpoints - [Get Account List - Isolated Margin - V1](https://www.kucoin.com/docs-new/abandoned-endpoints/margin-trading/get-account-list-isolated-margin-v1.md) | N/A (Margin **LF** isolated account list endpoint; no HF parameter mapping) | SPOT Margin LF endpoint is no longer available. |
| 10 | Abandoned Endpoints - [Get Account Detail - Isolated Margin - V1](https://www.kucoin.com/docs-new/abandoned-endpoints/margin-trading/get-account-detail-isolated-margin-v1.md) | N/A (Margin **LF** isolated account detail endpoint; no HF parameter mapping) | SPOT Margin LF endpoint is no longer available. |

**Developer Action Items (Recommended)**
- Use non-V2 account type parameters: `MARGIN`, `ISOLATED`.
- Keep compatibility for `*_V2` parameters for now, but plan to remove dependencies as they will be gradually replaced by non-V2 parameters.
- For Classic REST - Flex Transfer: use `MARGIN`, `ISOLATED` for non-`INTERNAL` transfers; V2 is only equivalent when `type=INTERNAL`.
- For Abandoned Endpoints - Inner Transfer: parameters must be lowercase: `margin`, `isolated`, `margin_v2`, `isolated_v2`.
- Stop calling disabled Margin LF order endpoints (Abandoned Endpoints).

### 2026.02.15
[Exchange Broker Modify] [Add sub-account API](https://www.kucoin.com/docs-new/rest/broker/exchange-broker/add-subaccount-api.md), [Modify sub-account API](https://www.kucoin.com/docs-new/rest/broker/exchange-broker/modify-subaccount-api.md) endpoints have been updated. The **permissions** field now includes four new enum permissions: **Margin**, **InnerTransfer** (Universal Transfer), **Transfer** (Withdrawal), and **Earn**.  

With the updated permission structure, the special sub-accounts of exchange brokers now additionally support:
- Margin Trading,
- Fund transfers between different types of accounts(please note: only transfer-within-account is currently supported),
- Direct withdrawals from sub-accounts,
- Earn (wealth management) services
 
At the same time,the KuCoin Convert feature is now accessible to special sub-accounts, provided that the broker has activated the Convert permission on their main account and the API permission of sub-accounts includes Spot.Learn more about KuCoin Convert: https://www.kucoin.com/convert

Existing integrations remain compatible. However, exchange brokers wishing to enable the new features must explicitly configure the corresponding permissions via Modify sub-account API endpoint.

We are excited to announce the launch of public interfaces for the Unified Trading Account API on September 29, 2025, at 08:00 UTC. These interfaces provide a streamlined way to access trading and market data under the Unified Trading Account framework. The specific changes are as follows:
Domain Unification: All API requests for the Unified Trading Account will use the unified domain: https://api.kucoin.com.

1. [Get Announcements](https://www.kucoin.com/docs-new/rest/ua/get-announcements.md)
  - Description: Obtain the latest news announcements, with default page search for announcements within a month.
  - Replaces: GET /api/v3/announcements
2. [Get Currency](https://www.kucoin.com/docs-new/rest/ua/get-currency.md)
  - Description: Request the currency details of a specified currency.
  - Replaces: 
    - GET /api/v3/currencies/{currency}
    - GET /api/v3/currencies
3. [Get Symbol](https://www.kucoin.com/docs-new/rest/ua/get-symbol.md)
  - Description: Request a list of available currency pairs for trading.
  - Replaces: 
    - SPOT: GET /api/v2/symbols/{symbol}
    - SPOT: GET /api/v2/symbols
    - FUTURES: GET /api/v1/contracts/{symbol}
    - FUTURES: GET /api/v1/contracts/active
4. [Get Ticker](https://www.kucoin.com/docs-new/rest/ua/get-ticker.md)
  - Description: Request market tickers for all the trading pairs in the market (including 24h volume); takes a snapshot every 2 seconds.
  - Replaces: 
    - SPOT: GET /api/v1/market/orderbook/level1
    - SPOT: GET /api/v1/market/allTickers
    - FUTURES: GET /api/v1/ticker
    - FUTURES: GET /api/v1/allTickers
5. [Get Trades](https://www.kucoin.com/docs-new/rest/ua/get-trades.md)
  - Description: Get the latest 100 trades of the specified symbol.
  - Replaces: 
    - SPOT: GET /api/v1/market/histories
    - FUTURES: GET /api/v1/trade/history
6. [Get OrderBook](https://www.kucoin.com/docs-new/rest/ua/get-orderbook.md)
  - Description: Query order book depth data (aggregated by price). Suitable for professional traders; use WebSocket incremental feed for updates due to strict rate limits.
  - Replaces: 
    - SPOT: GET /api/v1/market/orderbook/level2_{size}
    - SPOT: GET /api/v3/market/orderbook/level2
    - FUTURES: GET /api/v1/level2/depth{size}
    - FUTURES: GET /api/v1/level2/snapshot
7. [Get Klines](https://www.kucoin.com/docs-new/rest/ua/get-klines.md)
  - Description: Get the Kline (candlestick) data of the symbol, returned in grouped buckets based on requested type. Returns up to 1500 pieces of data per query; page by time for more data.
  - Replaces: 
    - SPOT: GET /api/v1/market/candles
    - FUTURES: GET /api/v1/kline/query
8. [Get Current Funding Rate](https://www.kucoin.com/docs-new/rest/ua/get-current-funding-rate.md)
  - Description: Get the current futures funding fee rate.
  - Replaces: GET /api/v1/funding-rate/{symbol}/current
9. [Get History Funding Rate](https://www.kucoin.com/docs-new/rest/ua/get-history-funding-rate.md)
  - Description: Query the funding rate at each settlement time point within a certain time range for the corresponding contract.
  - Replaces: GET /api/v1/contract/funding-rates
10. [Get Cross Margin Config](https://www.kucoin.com/docs-new/rest/ua/get-cross-margin-config.md)
  - Description: Request the configuration info of cross-margin trading.
  - Replaces: GET /api/v1/margin/config
11. [Get Service Status](https://www.kucoin.com/docs-new/rest/ua/get-service-status.md)
  - Description: Get the service status.
  - Replaces: 
    - SPOT: GET /api/v1/status
    - FUTURES: GET /api/v1/status

We will continue to roll out additional Unified Trading Account API interfaces in Q4 2025. Please refer to future announcements for updates.
